# Arduino-TicTacToe
Play TicTacToe against a smart AI \
Because of the limitation of an 8 * 8 Field instead of a cross and circles I use squares and lines. \

## Components
Elegoo UNO R3\
MAX7219\
Membrane Switch Module 4 * 4\
